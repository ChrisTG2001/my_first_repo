# Nombre: PLATZIVIDEO
*Descripción*:

Platzi videos es una plataforma que te permite aprender diferentes tipos de lenguajes de programación

*Instalación*:

puedes instalarlo desde npm

npm install platzi-video

*o también clonando el repositorio*

git clone url

import platzivideo

video = platzivideo()
